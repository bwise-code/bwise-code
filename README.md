## About Me

I'm Brian — a board-certified psychiatrist and developer with a passion for automation. By day, I work at the intersection of mental health, integrative medicine, and patient care. By night, I build smart bots and automation tools that make life a little easier.

I believe the same curiosity that drives good medicine drives good code: understand the problem deeply, then build the simplest solution that actually works.

- 🤖 Building a **Telegram Bot** (Python) and a **Discord Bot** (Node.js)
